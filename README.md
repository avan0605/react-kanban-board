# React Kanban Board

A Trello-style Kanban task management application built using React.

## Features
- Create tasks
- Organize tasks into columns (To Do, In Progress, Done)
- Context API state management
- Reusable component architecture
- Clean modular structure

## Tech Stack
- React
- Context API
- CSS Modules

## Architecture
- Global state handled using Context API
- Component hierarchy:
  App → Board → Column → Card

## Future Improvements
- Drag and drop support
- Persistent storage
- Backend integration
